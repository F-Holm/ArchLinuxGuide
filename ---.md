## Setup

- Descargar qbittorrent
- Descargar imagen ISO mediante torrent
- Ya se puede desinstalar qbittorrent
- Descargar o copiar PGP signature
- verificar (se puede hacer desde WSL)
- Preparar pendrive con rufus y la imagen ISO

## Boot

- Iniciar la PC con el pendrive

### internet y teclado

- loadkeys la-latin1
- localectl list-keymaps

- iwctl
- device list
- station "name" scan
- station "name" get-networks
- station "name" connect "SSID"
- ip link
- ping ping.archlinux.org

### Particiones

- fdisk -l
- cfdisk "dir disk (Ej: /dev/dva)"
- 1 GB para boot de tipo `EFI System`
- (no recomendado) 2*RAM B para swap de tipo `Linux swap`
- restante para boot de tipo `Linux filesystem`
- fdisk -l
- cryptsetup luksFormat "dir partición principal"
- cryptsetup open "dir partición principal" cryptroot
- mkfs.ext4 "dir partición principal encriptada"
- mkswap "dir particion swap (si existe)"
- mkfs.fat -F 32 "dir partición boot"
- fdisk -l
- mount "dir particion principal encriptada" /mnt
- mount --mkdir "dir particion boot" /mnt/boot
- swapon "dir particion swap (si existe)"

### Install

- pacstrap -K /mnt base linux linux-firmware
- genfstab -U /mnt >> /mnt/etc/fstab
- arch-chroot /mnt
- fallocate -l 4G /swapfile
<br>
Recomendado tener el doble de tamaño de la memoria ram como tamaño de swap
- chmod 600 /swapfile
- mkswap /swapfile
- swapon /swapfile
- pacman -S nano grub efibootmgr

### config

- nano /etc/mkinitcpio.conf
<br>
HOOKS=(base udev autodetect modconf block keyboard keymap encrypt filesystems)

- mkinitcpio -P
- options cryptdevice=UUID=<uuid_de_sda3>:cryptroot root=/dev/mapper/cryptroot rw
- ln -sf /usr/share/zoneinfo/America... /etc/localtime
- hwclock --systohc
- nano /etc/locale.gen
<br>
descomentar linea correspondiente
- locale-gen
- nano /etc/locale.conf
<br>
LANG=lenguaje puesto en el archivo anterior
- nano /etc/hostname
<br>
"nombre"
- passw
- useradd -m  "nombre usuario"
- passwd "nombre usuario"
- grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB
- grub-mkconfig -o /boot/grub/grub.cfg
- pacman -S "intel-ucode o amd-ucode"
- pacman -S networkmanager
- systemctl enable NetworkManager.service
- nano /etc/sudoers
<br>
copiar linea de privilegios de root y pegarla con el nombre de usuario deseado en vez de root dejando en la linea original los permisos de root
- reboot
